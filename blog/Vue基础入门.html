Vue基础入门

指令

var app = new Vue({
    // 绑定id
    el: '#app',

    // 数据和元素属性:v-for v-bind
    data: {
        name: '',
        age: ''
    },

    // 事件绑定:v-on
    methods: {

    },

    // 计算属性:
    computed:{
        sum:function(){
            return this.math + this.xxx + this.eng
        },
        average:function(){
            return this.sum / 3
        }
    },
    props:[]
    ,
})
  • v-for
    • 用于迭代的指令（动态监控数据变化）
  • v-bind
    • 用于绑定数据和元素属性的指令（双向绑定）
    • 还可以绑定类
    • 通俗写法v-bind:href="xx"、v-bind:class="xx"
    • 快捷写法:class="xx"、:img="xx"
  • v-on
    • 用于绑定事件
    • 通俗写法：:
      • v-on:click="onClick"
      • v-on="{mouseenter: onEnter, mouseleave: onOut}"
      • v-on:submit.prevent="onSubmit" 阻止整页刷新
      • v-on:submit.stop="onSubmit" 阻止冒泡事件
      • v-on:keyup.enter="onKEnter" 监测回车键
    • 快捷写法：@
      • @click="onClick"
      • @"{mouseenter: onEnter, mouseleave: onOut}"
      • @submit.prevent="onSubmit" 阻止整页刷新
      • @submit.stop="onSubmit" 阻止冒泡事件
      • @keyup.enter="onKEnter" 监测回车键
  • v-model（非常常用）
    • 好用、安全、强大
    • v-model.lazy 惰性更新（输入完了，才绑定）
    • v-model.trim 去掉前后空格（密码输入时慎用）
    • v-model.number 字符串自动转数字类型、
    • 不同元素和类型上使用：
      • <input>：type=text、type=radio（单选）、type=checkbox（复选）
      • <textarea>：默认内容
      • <select>：
    • 控制流指令
      • v-if= "role == 'admin' " // 管理员你好
      • v-else-if = "role == 'hr' // HR你好
      • v-else // 你没有权限访问

计算属性

  • computed:{}：用于通过其他数据，算出新数据，可以把结果进行缓存，其他数据没变的情况下，调用缓存的数据，提高性能

组件（可维护性，可重用性）

  • 全局及局部组件

    • Vue.component('alert',{...})

      //JS内部
Vue.component('alert',{
    template: '<button @click="onClick">弹窗</button>'，
    methods:{
        onClick:function(){
            alert('OJBK!');
        }
    }
}); 
// 全局

new Vue({
    el:'#app',
    components:{
        'alert':{
            template: '<button @click="onClick">弹窗</button>',
            methods:{
                onClick:function(){
                                alert('OJBK!');
                            }
                }
            }
})
// 局部

//HTML内部
<div id="app">
    <alert></alert> // 直接调用上面功能
</div>
